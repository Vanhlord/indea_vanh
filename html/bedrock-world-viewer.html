Kéo world vào và đọc bản đồ 2D theo kiểu Bedrock.

Trang này là điểm bắt đầu cho Bedrock world viewer trong web. Bạn có thể kéo thả folder world hoặc file .mcworld, hệ thống sẽ kiểm tra cấu trúc gói world, đọc nhanh các dấu hiệu quan trọng như db/ và level.dat, rồi dựng một preview 2D để chuẩn bị cho pipeline render thật.

Workspace kéo thả

Ưu tiên kéo cả folder world đã giải nén để trình duyệt nhìn thấy cấu trúc tương đối. Bạn vẫn có thể chọn file .mcworld, nhưng bước parse archive sẽ là phần tiếp theo của pipeline.

Step 01 Nhận package world

Đọc file list, tên world, mode folder hay archive và root path tương đối.

Step 02 Phát hiện Bedrock signals

Tìm db/, level.dat, levelname.txt và dấu hiệu chunk store.

Step 03 Chunk scan chuẩn bị render

Ở bước sau sẽ parse LevelDB thật để lấy block top layer, biome và palette.

Step 04 Xuất map 2D

Pan, zoom, tile cache, đánh dấu spawn, điểm quan tâm và nhiều dimension.
