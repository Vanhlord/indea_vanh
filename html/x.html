Tải video từ X

Dán link bài đăng X/Twitter, xem nhanh thông tin và tải video về máy bằng một nút bấm.

Hỗ trợ `x.com` Hỗ trợ `twitter.com` Hỗ trợ `t.co`

Preview

Thông tin bài đăng

Chưa có dữ liệu

Nhập link X rồi bấm “Phân tích link”.

Nhanh

Backend dùng `yt-dlp` để lấy video trực tiếp từ bài đăng.

Gọn

Chỉ cần dán link, hệ thống tự nhận diện và tải file MP4.

Ổn định

Hỗ trợ link `x.com`, `twitter.com` và `t.co` phổ biến.

Cách dùng

1. Copy link bài đăng

Mở X/Twitter và sao chép link của bài đăng có video.

2. Phân tích

Dán link vào ô nhập và bấm “Phân tích link” để tải metadata.

3. Xem preview

Kiểm tra thumbnail hoặc video preview trước khi tải xuống.

4. Tải về

Bấm nút tải xuống để lưu file MP4 vào máy bạn.
