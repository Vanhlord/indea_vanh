Template Code (views/example.ejs):

   <h1><%= title %></h1>              ← Output variable
   
   <% if (user) { %>                 ← Condition (no output)
       <p>Welcome <%= user.name %></p>
   <% } %>
   
   <ul>
   <% items.forEach(item => { %>     ← Loop
       <li><%= item.title %></li>
   <% }); %>
   </ul>

Route Handler (server-side):

   app.get('/example', async (req, res) => {
       const data = {
           title: 'My Page',
           user: { name: 'John' },
           items: [{title: 'Item 1'}, {title: 'Item 2'}]
       };
       res.render('example', data);
   });

Browser Receives:

   <h1>My Page</h1>
   <p>Welcome John</p>
   <ul>
       <li>Item 1</li>
       <li>Item 2</li>
   </ul>
